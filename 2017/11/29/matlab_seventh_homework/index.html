y_ = beta(1) + beta(2)*x_mesh1 + beta(3)*x_mesh2 + beta(4)*(x_mesh1.^2) + beta(5)*(x_mesh2.^2) + beta(6)*x_mesh1.*x_mesh2;

下面用另一种方法求最大值，matlab中提供了fminsearch()这个函数求最小值，将原函数前面加个负号，就可以通过fminsearch()来求解了。

f_beta = @(x)(-(beta(1) + beta(2)*x(1) + beta(3)*x(2) + beta(4)*x(1)^2 + beta(5)*x(2)^2 + beta(6)*x(1)*x(2)))
